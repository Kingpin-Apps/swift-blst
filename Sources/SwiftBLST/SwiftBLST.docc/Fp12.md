# ``SwiftBLST/Fp12``

An element of the GT group — the output of a BLS12-381 pairing.

## Overview

`Fp12` is a 576-byte structure representing an element of the degree-12 extension field of the BLS12-381 base field. It is the target group GT of the pairing map e: G1 × G2 → GT.

You do not construct `Fp12` values directly — they are produced by ``Pairing/millerLoop(g1:g2:)``.

```swift
// Produce GT elements
let e1 = Pairing.millerLoop(g1: p, g2: q)
let e2 = Pairing.millerLoop(g1: r, g2: s)

// Multiply GT elements (for batch verification)
let product = e1 * e2

// Compare pairings
let same = Pairing.finalVerify(e1, e2)
```

### Equality

`Fp12` conforms to `Equatable`, but direct `==` comparison on raw Miller loop outputs is **not** a correct pairing equality check. Two mathematically equal pairings may differ as raw `Fp12` values before final exponentiation. Use ``Pairing/finalVerify(_:_:)`` for all production equality checks.

Direct `==` is useful for testing intermediate values or comparing final-exp outputs.
